# My Vue 3 Skill

自定义的 Vue 3 组件生成技能。

## 安装

```bash
npx skills add path/to/my-skill
```

## 使用

安装后，AI 助手将能够：
- 生成符合最佳实践的 Vue 3 组件
- 使用 TypeScript 定义类型
- 遵循代码规范和组件结构
- 应用常见的设计模式

## 技能内容

- Vue 3 Composition API 最佳实践
- TypeScript 类型定义规范
- 组件生命周期管理
- 性能优化技巧
- 常见模式和代码示例
