📋 环境变量说明
变量名 说明 示例 / 备注
MINIMAX_API_KEY MiniMax API 密钥（用于 VL 视觉定位 + 图像编辑） 必填 eyJ…
MINIMAX_BASE_URL MiniMax API 地址 https://api.minimax.chat/v1（默认）
MINIMAX_VISION_PLAN_ENABLED 启用 MiniMax VL 视觉定位，规划替换区域 设为 1 启用
MINIMAX_IMAGE_EDIT_ENABLED 启用 MiniMax 图像编辑（产品替换） 设为 1 启用
MINIMAX_IMAGE_MODEL 图像生成 / 编辑模型 image-01（默认）
PADDLEOCR_ACCESS_TOKEN PaddleOCR Cloud 访问令牌（文字识别） 推荐 字符串
SAM_CHECKPOINT Segment Anything 模型权重路径（本地 SAM 分割） 推荐 ./checkpoints/sam_vit_b.pth
SAM2_MODEL_CFG / SAM2_CHECKPOINT SAM2 模型配置 & 权重路径 可选
IMAGE_EDIT_PROVIDER 图像编辑 Provider minimax_images | custom_http | none
CUSTOM_HTTP_ENDPOINT 自定义图像编辑 HTTP 服务地址 可选
PIXELFORGE_PORT Web UI 监听端口 8700（默认）
PIXELFORGE_OUTPUT_DIR 项目输出根目录 ./pixelforge_runs（默认）
PIXELFORGE_LOG_LEVEL 日志级别 INFO | DEBUG | WARNING
🔧 配置步骤
# 1. 复制示例配置文件
# ───────────────────────────────────────────
# 从项目根目录复制 .env.example 为 .env
# cp .env.example .env      # Linux/macOS
# copy .env.example .env    # Windows

# 2. 编辑 .env，填入你的密钥
# ───────────────────────────────────────────
MINIMAX_API_KEY=***
MINIMAX_VISION_PLAN_ENABLED=1
MINIMAX_IMAGE_EDIT_ENABLED=1
PADDLEOCR_ACCESS_TOKEN=your-paddleocr-token

# 3. 重启 Web UI 使配置生效
# ───────────────────────────────────────────
# uvicorn pixelforge.api.main:app --port 8700 --reload

# 4. 如需 SAM 分割，手动下载权重
# ───────────────────────────────────────────
# https://github.com/facebookresearch/segment-anything#model-checkpoints
