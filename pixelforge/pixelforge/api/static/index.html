快速上手

通过命令行或 Web UI 两种方式使用 PixelForge

命令行

# ── 安装 ──────────────────────────────────
pip install -e .

# ── 分析图片（图层拆分） ───────────────────
python -m pixelforge analyze -i ./photo.jpg -o ./output

# ── SAM 分割 ──────────────────────────────
python -m pixelforge segment auto -i ./photo.jpg -c sam_vit_b.pth

# ── 启动 Web UI ───────────────────────────
uvicorn pixelforge.api.main:app --port 8700 --reload

# ── 查看配置 ──────────────────────────────
python -m pixelforge settings
