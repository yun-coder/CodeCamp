# hiprint 自定义属性配置说明

## 概述

在 hiprint 中，属性设置面板是通过 `new window.hiprint.PrintTemplate` 初始化时绑定 `settingContainer` 来显示的。本文档将详细说明如何手动自定义元素的属性。

## 当前实现

在 `printDesigner.vue` 中，模板初始化时绑定了属性容器：

```javascript
hiprintTemplate = new window.hiprint.PrintTemplate({
  template: getCustomPrintJson(),
  settingContainer: '#PrintElementOptionSetting',  // 绑定属性容器
  paginationContainer: '.hiprint-printPagination',
});
hiprintTemplate.design('#hiprint-printTemplate');
```

属性面板容器 `#PrintElementOptionSetting` 在 `settingsPanel.vue` 中定义：

```vue
<div id="PrintElementOptionSetting"></div>
```

## 自定义属性的方式

### 方式一：通过全局配置扩展（推荐）

通过配置 `window.HIPRINT_CONFIG` 来注册自定义属性项：

```javascript
// 在 hiprint 初始化之前配置
window.HIPRINT_CONFIG = {
  optionItems: [
    {
      name: 'myCustomOption',  // 自定义属性名称
      title: '我的自定义属性',  // 显示标题
      type: 'input',  // 类型：input/select/checkbox/color/number
      defaultValue: '',
      onChange: function(value, element) {
        // 属性值变化时的回调
        console.log('属性改变:', value);
      }
    },
    {
      name: 'mySelectOption',
      title: '下拉选择',
      type: 'select',
      options: [
        { value: 'option1', label: '选项1' },
        { value: 'option2', label: '选项2' }
      ],
      defaultValue: 'option1'
    }
  ]
};

// 然后初始化 hiprint
window.hiprint.init({
  providers: [new customElementTypeProvider()]
});
```

### 方式二：在元素类型定义时配置属性

在定义元素类型时，通过 `options` 对象配置支持的属性：

```javascript
// 在 hiprintConfig.js 中定义元素类型
{
  tid: 'testModule.customElement',
  text: '自定义元素',
  type: 'text',
  data: '',
  options: {
    // 基础属性
    width: 100,
    height: 30,
    
    // 自定义属性
    myCustomField: '默认值',
    customColor: '#1890ff',
    
    // 样式属性
    fontSize: 12,
    fontFamily: 'Microsoft YaHei',
    textAlign: 'center',
    
    // 特殊配置
    formatter: function(value, options) {
      // 自定义格式化
      return value;
    }
  }
}
```

### 方式三：创建完全自定义的元素类型

创建一个完全自定义的元素类型，包含自定义属性配置：

```javascript
// 1. 创建自定义元素类型提供者
export const myCustomProvider = function () {
  const addElementTypes = function (context) {
    context.addPrintElementTypes(
      "myModule",
      [
        new window.hiprint.PrintElementTypeGroup("我的模块", [
          {
            tid: 'myModule.specialElement',
            text: '特殊元素',
            type: 'text',
            data: '',
            options: {
              // 支持的属性选项
              width: 150,
              height: 40,
              fontSize: 14,
              fontWeight: 'bold',
              textAlign: 'left',
              
              // 自定义属性
              customAttribute1: 'value1',
              customAttribute2: true,
              
              // 格式化函数
              formatter: function(data, options) {
                // 自定义数据格式化逻辑
                return `自定义: ${data}`;
              },
              
              // 样式器
              styler: function(data, options, $target) {
                // 返回自定义样式对象
                return {
                  'background-color': options.customColor || '#fff',
                  'border': '1px solid #ccc'
                };
              }
            }
          }
        ])
      ]
    );
  };

  return {
    addElementTypes: addElementTypes
  };
};

// 2. 注册自定义提供者
window.hiprint.init({
  providers: [
    new customElementTypeProvider(),  // 原有的
    new myCustomProvider()  // 新增的
  ]
});
```

### 方式四：扩展现有元素的属性配置

如果需要为现有元素类型添加额外的属性支持，可以通过配置 `supportOptions`：

```javascript
// 配置支持的属性项
window.HIPRINT_CONFIG = {
  // 定义可用的属性项
  optionItems: [
    {
      name: 'seamless',
      title: '骑缝章',
      type: 'checkbox',
      defaultValue: false
    },
    {
      name: 'acrossPages',
      title: '跨页显示',
      type: 'checkbox',
      defaultValue: false
    },
    {
      name: 'customColor',
      title: '自定义颜色',
      type: 'color',
      defaultValue: '#000000'
    }
  ],
  
  // 配置元素类型的支持属性
  image: {
    supportOptions: [
      { name: 'width' },
      { name: 'height' },
      { name: 'src' },
      { name: 'seamless' },  // 添加自定义属性
      { name: 'acrossPages' },
      { name: 'customColor' }
    ]
  }
};
```

## 实际应用示例

### 示例1：为骑缝章添加特殊属性

```javascript
// 在 hiprintConfig.js 中
{
  tid: 'stampModule.seamlessStamp',
  text: '骑缝章',
  type: 'image',
  data: '',
  options: {
    // 基础属性
    width: 50,
    height: 50,
    
    // 骑缝章特殊属性
    seamless: true,           // 标识为骑缝章
    acrossPages: true,        // 跨页显示
    stampConfig: {
      pageCount: 3,           // 切割页数
      overlapWidth: 10,       // 重叠宽度
      direction: 'vertical'   // 切割方向
    },
    
    // 位置控制
    lock: false,
    hidden: false,
    
    // 自定义渲染
    formatter: function(data, options) {
      // 根据配置返回对应的切割图片
      const { stampConfig, field } = options;
      if (stampConfig) {
        return `/api/stamp/split?pages=${stampConfig.pageCount}`;
      }
      return data;
    }
  }
}
```

### 示例2：创建带验证的自定义属性

```javascript
window.HIPRINT_CONFIG = {
  optionItems: [
    {
      name: 'customUrl',
      title: '自定义链接',
      type: 'input',
      defaultValue: '',
      validate: function(value) {
        // 验证 URL 格式
        const urlPattern = /^https?:\/\/.+/;
        if (!urlPattern.test(value)) {
          alert('请输入有效的 URL');
          return false;
        }
        return true;
      },
      onChange: function(value, element) {
        console.log('URL 改变为:', value);
        // 可以在这里触发其他操作
      }
    },
    {
      name: 'maxLength',
      title: '最大长度',
      type: 'number',
      defaultValue: 100,
      min: 1,
      max: 1000,
      step: 1
    }
  ]
};
```

### 示例3：完全自定义属性面板

如果 hiprint 默认的属性面板不满足需求，可以手动创建自定义面板：

```vue
<!-- settingsPanel.vue -->
<template>
  <div class="settings-panel">
    <a-typography-title :level="5">属性设置</a-typography-title>
    
    <!-- hiprint 自动生成的属性 -->
    <div id="PrintElementOptionSetting"></div>
    
    <!-- 自定义扩展属性 -->
    <div v-if="selectedElement" class="custom-options">
      <a-divider>扩展属性</a-divider>
      
      <a-form layout="vertical">
        <a-form-item label="骑缝章页数">
          <a-input-number 
            v-model:value="customOptions.pageCount"
            :min="1"
            :max="10"
            @change="updateElement"
          />
        </a-form-item>
        
        <a-form-item label="重叠宽度(px)">
          <a-slider 
            v-model:value="customOptions.overlapWidth"
            :min="0"
            :max="50"
            @change="updateElement"
          />
        </a-form-item>
        
        <a-form-item label="切割方向">
          <a-radio-group 
            v-model:value="customOptions.direction"
            @change="updateElement"
          >
            <a-radio value="vertical">垂直</a-radio>
            <a-radio value="horizontal">水平</a-radio>
          </a-radio-group>
        </a-form-item>
      </a-form>
    </div>
  </div>
</template>

<script setup>
import { ref, watch } from 'vue';

const selectedElement = ref(null);
const customOptions = ref({
  pageCount: 3,
  overlapWidth: 10,
  direction: 'vertical'
});

// 监听元素选择
const watchElementSelection = () => {
  // 通过 jQuery 事件监听元素选择
  $(document).on('PrintElementSelectEvent', function(event) {
    selectedElement.value = event.target;
    // 读取当前元素的自定义属性
    if (event.target.options) {
      customOptions.value = {
        pageCount: event.target.options.pageCount || 3,
        overlapWidth: event.target.options.overlapWidth || 10,
        direction: event.target.options.direction || 'vertical'
      };
    }
  });
};

// 更新元素属性
const updateElement = () => {
  if (selectedElement.value) {
    // 更新元素的自定义属性
    Object.assign(selectedElement.value.options, customOptions.value);
    // 触发重新渲染
    selectedElement.value.refresh();
  }
};

onMounted(() => {
  watchElementSelection();
});
</script>
```

## 属性配置项说明

### 基础配置字段

| 字段名 | 类型 | 说明 |
|--------|------|------|
| `name` | string | 属性名称（唯一标识） |
| `title` | string | 属性显示标题 |
| `type` | string | 属性类型：input/select/checkbox/color/number/textarea |
| `defaultValue` | any | 默认值 |
| `hidden` | boolean | 是否隐藏该属性 |

### 扩展配置字段

| 字段名 | 类型 | 说明 |
|--------|------|------|
| `options` | array | select 类型的选项列表 |
| `min` | number | number 类型的最小值 |
| `max` | number | number 类型的最大值 |
| `step` | number | number 类型的步长 |
| `validate` | function | 验证函数 |
| `onChange` | function | 值变化回调 |
| `formatter` | function | 数据格式化函数 |
| `styler` | function | 样式生成函数 |

## 注意事项

1. **初始化顺序**：必须在 `hiprint.init()` 之前配置 `window.HIPRINT_CONFIG`

2. **属性名称**：自定义属性名称应避免与内置属性冲突（如 width、height、left、top 等）

3. **属性持久化**：自定义属性会自动保存在模板 JSON 中，无需额外处理

4. **类型支持**：不同元素类型支持的属性不同，需要根据具体类型配置

5. **事件监听**：可以通过 jQuery 事件监听元素选择和属性变化：
   ```javascript
   $(document).on('PrintElementSelectEvent', function(e) {
     // 元素被选中
   });
   
   $(document).on('PrintElementOptionChangeEvent', function(e) {
     // 属性发生变化
   });
   ```

6. **动态更新**：修改元素属性后，需要调用 `element.refresh()` 或 `hiprintTemplate.update()` 来更新显示
