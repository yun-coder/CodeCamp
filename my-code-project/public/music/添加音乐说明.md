# 如何从酷狗音乐获取《爱很简单》文件

## 方法一：使用酷狗电脑客户端

1. **下载并安装酷狗音乐客户端**
   - 访问 https://www.kugou.com/
   - 下载并安装酷狗音乐PC版

2. **登录会员账号**
   - 打开酷狗音乐
   - 登录您的会员账号

3. **搜索并下载《爱很简单》**
   - 在搜索框输入"陶喆 爱很简单"
   - 找到原版（非remix、非DJ版本）
   - 右键点击歌曲，选择"下载"
   - 选择音质（建议选择标准或高品质）
   - 下载完成

4. **移动到项目文件夹**
   - 找到下载的mp3文件
   - 重命名为 `love-simple.mp3`
   - 移动到 `public/music/` 文件夹

## 方法二：使用酷狗音乐网页版

1. **访问酷狗音乐网页版**
   - 访问 https://www.kugou.com/

2. **搜索歌曲**
   - 搜索"陶喆 爱很简单"
   - 找到原版

3. **登录会员并下载**
   - 登录您的会员账号
   - 点击下载按钮
   - 选择下载路径

4. **移动到项目文件夹**
   - 重命名为 `love-simple.mp3`
   - 移动到 `public/music/` 文件夹

## 方法三：通过手机酷狗APP

1. **打开手机酷狗APP**
   - 登录您的会员账号

2. **搜索并下载**
   - 搜索"陶喆 爱很简单"
   - 找到原版并点击下载

3. **传输到电脑**
   - 通过数据线或云盘传输到电脑
   - 重命名为 `love-simple.mp3`
   - 移动到 `public/music/` 文件夹

## 添加音乐文件后的步骤

1. **创建文件夹**（如果不存在）
   ```bash
   mkdir -p public/music
   ```

2. **将文件放入文件夹**
   - 将下载的 mp3 文件命名为 `love-simple.mp3`
   - 放到 `public/music/` 文件夹

3. **修改代码**
   在 `src/components/ProposalPage.vue` 中修改 `musicUrls`：

   ```javascript
   musicUrls: [
     '/music/love-simple.mp3', // 您的音乐文件
     'https://www.soundhelix.com/examples/mp3/SoundHelix-Song-1.mp3' // 备用
   ]
   ```

4. **重启开发服务器**
   ```bash
   # 停止当前服务器 (Ctrl+C)
   npm run dev
   ```

5. **测试播放**
   - 刷新浏览器
   - 点击页面任意位置或右上角音乐按钮
   - 检查音乐是否正常播放

## 常见问题

**Q: 为什么放在 public 文件夹而不是 src？**
A: public 文件夹的文件会被直接复制到构建输出目录，可以直接通过 URL 访问，不需要经过 webpack 处理。

**Q: 文件名必须是 love-simple.mp3 吗？**
A: 不是必须的，但建议使用英文和数字，避免中文可能导致的路径问题。修改文件名后记得同时修改代码中的路径。

**Q: 可以使用其他音乐文件吗？**
A: 可以！任何 mp3 格式的音乐文件都可以，只需修改代码中的文件名即可。
