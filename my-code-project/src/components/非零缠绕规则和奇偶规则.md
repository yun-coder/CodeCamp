#   在图形学中判断一个点是否在多边形内
    1、多边形不是自相交

    2、多边形自相交（多边形在平面内除顶点外还有其他公共点）
        2.1、奇偶规则 Odd-even rule
            （奇内偶外）
            奇数表示在多边形内，偶数表示在多边形外
            说明：从任意位置P作一条射线，若与该射线相交的多边形的数目为奇数，则P是多边形内部点，否则是外部点。
        2.2、非零环绕数规则 Non-zero widing Number rule
            零为内，非零为外
            说明：多边形的边为矢量。将环绕数初始化为零。再从任意位置P作一条射线。
            当从P点沿射线方向移动时，对在每个方向上穿过射线的边计数，
            每当多边形的边从 右到左 穿过射线时 +1
            每当多边形的边从 左到右 穿过射线时 -1
